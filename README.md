
# Parkinson's Disease Detection using Machine Learning

This project detects Parkinson’s disease using multiple modalities:

1. Speech Analysis
2. Spiral Drawing Analysis
3. Handwriting Recognition

Technologies Used:
- Python
- TensorFlow
- Keras
- OpenCV
- Gradio

Features:
✔ Speech based detection
✔ Spiral drawing detection
✔ Handwriting detection
✔ Real-time prediction
